## Quick start

### Project setup

```bash
$ docker compose build
```

### Compile and run the project

```bash
# sever started in watch mode by default
$ docker compose up
```

In docker container, start the server in other modes:

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
